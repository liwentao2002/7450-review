## Recommended Study Workflow

Use the materials in this order:

1. Read the relevant lecture slides.
2. Read the matching unit in `INFS7450_Study_Guide_Week2-7.md`.
3. Memorise the key vocabulary and formulas.
4. Work through the example calculations.
5. Attempt related past exam questions.
6. Record mistakes in a separate error log.
7. Update the study guide when new materials are added.

中文学习顺序：

1. 先看对应 lecture slides。
2. 再读复习讲义里的对应单元。
3. 背重点词汇和公式。
4. 跟着例题演示练公式。
5. 做对应历年考试题。
6. 把错题记录到错题本。
7. 新材料加入后继续更新复习讲义。

中文：Week 2 到 Week 7 的核心目标是把图计算、中心性、网络性质、同配性和信息扩散这些基础内容练到可以手算和英文解释。

## Exam Preparation Strategy

The final exam is expected to focus on written calculations and conceptual explanations rather than coding.

High-score preparation should focus on:

- calculating graph metrics by hand
- tracing algorithms step by step
- explaining models in clear English
- comparing similar concepts
- recognising repeated exam patterns
- adapting to new questions from later lectures

中文：期末重点预计是手算和概念解释，不是现场写代码。高分关键是会手算图指标、逐步推算法、用英文解释模型、比较相似概念，并能根据后续 lecture 适应新题。

## Update Rules

When new materials are added, update the study guide instead of creating scattered notes.

Suggested update process:

1. Add the new PDF or quiz file to this folder.
2. Extract the main topics and key vocabulary.
3. Add a new unit or extend the Week 8 to Week 13 section.
4. Add formulas and worked examples if needed.
5. Map the new topic to past exam patterns.

中文：后续新增 Week 8 到 Week 13 的 PPT、quiz 或 tutorial 后，优先更新现有 study guide，不要把笔记分散到很多文件里。
